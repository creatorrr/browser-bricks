Browser Bricks

About.

Browser Bricks is a browser rendition of the famous Breakout game -- with a twist. Instead of the regular canvas, it's played using a collection of browser windows acting as bricks, paddle and ball. The empty space between windows becomes the playing field for the game!

Tribute to Browser Pong.

A departure from the typical browser gaming experience, this game was inspired by Browser Pong . In fact, it started life as a response to Stewart's letter-to-the-source-code-reader in the game source.

This game celebrates the spirit of openness that the web as a platform proliferates. Simplicity and openness were key goals of the project. The source code is (hopefully) very readable and doesn't use any external libraries. So hit View Source and take a look at the innards!

The entire codebase was released under the permissive MIT license . So you're free to reuse or remix any part of the project.

An experiment.

Let me make this extremely clear, this game is an experiment. It is a prototype at best so please forgive me for any rough edges. However, do feel free to shoot me an email or open a Github issue if you find any bugs or have any feedback.

Note on compatibility.

Given the vast inconsistencies in the way browsers deal with popups, there are numerous glitches on different platforms. In fact, this doesn't seem to work on Firefox at all. For now, it has only been tested on the latest version (v32+) of Chrome.
